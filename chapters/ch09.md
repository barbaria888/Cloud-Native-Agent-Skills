# Ch09 — Project 5: Frontend with Cloud Storage Static Hosting

## Architecture
```
Browser → Cloud Storage (static HTML/JS/CSS)
        → API calls → Global HTTP LB
                    → Cloud Run services (skill, fact, profile)
```

## Cloud Storage Static Website Hosting
```bash
# Make bucket public for website serving
gsutil iam ch allUsers:objectViewer gs://$BUCKET_NAME

# Set website config (main page + 404 page)
gsutil web set -m index.html -e 404.html gs://$BUCKET_NAME

# Upload site files
gsutil -m cp -r ./dist/* gs://$BUCKET_NAME

# Enable CORS (if API is on different domain)
gsutil cors set cors.json gs://$BUCKET_NAME
```

**cors.json:**
```json
[{
  "origin": ["https://yourdomain.com"],
  "method": ["GET", "POST", "PUT", "DELETE"],
  "responseHeader": ["Content-Type", "Authorization"],
  "maxAgeSeconds": 3600
}]
```

## Routing API Calls
- Frontend calls a unified API endpoint
- Global HTTP LB routes `/api/*` to the appropriate Cloud Run service
- Path-based routing: `/api/skills/*` → skill-service, `/api/facts/*` → fact-service, etc.

## Identity Platform Frontend Integration
```javascript
// Firebase SDK for browser-based auth
import { initializeApp } from 'firebase/app';
import { getAuth, signInWithEmailAndPassword, getIdToken } from 'firebase/auth';

const app = initializeApp(firebaseConfig);
const auth = getAuth(app);

// Sign in
const userCredential = await signInWithEmailAndPassword(auth, email, password);
const token = await getIdToken(userCredential.user);

// Use token in API calls
fetch('/api/facts', {
  headers: { 'Authorization': `Bearer ${token}` }
});
```

## API Gateway / Unified API
- Use Cloud Run URL rewrite rules via Global HTTP LB
- Or deploy a lightweight API gateway service on Cloud Run
- All microservices behind a single domain → avoids CORS issues

## Cloud Storage Signed URLs (for private content)
```bash
# Generate time-limited URL for private object
gsutil signurl -d 1h service_account.json gs://$BUCKET/$OBJECT
```

## Frontend Build Pattern
```bash
# Build frontend (React, Vue, etc.)
npm run build

# Upload to Cloud Storage
gsutil -m rsync -r -d ./dist gs://$BUCKET_NAME

# Invalidate CDN cache (if Cloud CDN enabled)
gcloud compute url-maps invalidate-cdn-cache $URL_MAP --path "/*"
```

## Cost Considerations
- Cloud Storage: $0.026/GB/month (hosting cost is minimal for static sites)
- Egress: free within same region; charged for cross-region/internet
- Cloud CDN: reduces latency globally; cache invalidation has a small cost
